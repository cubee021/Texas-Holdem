### 요약 내용

#### 1. GameMode vs GameState 역할 분리 명확화

**핵심 질문:**
- "베팅 처리 함수가 GameMode에 있어야 하나, GameState에 있어야 하나?"

**결론:**
- **GameMode (Server Only)**: 게임 **흐름 제어**
  - StartBettingRound(), IsBettingRoundComplete(), MoveToNextPlayer()
  - Phase 전환 로직
- **GameState (Replicated)**: 게임 **데이터 + 데이터 변경**
  - ProcessFold(), ProcessCheck(), ProcessCall(), ProcessRaise()
  - TotalPot, CurrentMaxBet, CurrentTurnPlayerIndex (Replicated 프로퍼티)
  - 데이터와 그 데이터를 다루는 함수는 함께 있는 게 자연스러움

**연결 흐름:**
```
[Client] E키 → RPC → [Server] PlayerController
  → GameState->Process*() (데이터 변경)
  → GameMode->MoveToNextPlayer() (흐름 제어)
  → if 베팅 완료: 다음 Phase로
```

#### 2. GameState 베팅 처리 함수 구현 (5개)

**HoldemGameState.h/cpp 추가:**

```cpp
void ProcessFold(AHoldemPlayerState* Player);
void ProcessCheck(AHoldemPlayerState* Player);
void ProcessCall(AHoldemPlayerState* Player);
void ProcessRaise(AHoldemPlayerState* Player);
void CollectBetsIntoPot();
```

**주요 로직:**
- **ProcessFold**: `bIsFolded = true`, `bHasActedThisRound = true`
- **ProcessCheck**: `bHasActedThisRound = true` (베팅액 변경 없음)
- **ProcessCall**:
  - `CallAmount = CurrentMaxBet - Player->CurrentBet` (이미 낸 금액 제외)
  - `CurrentChips -= CallAmount`
  - `CurrentBet = CurrentMaxBet`
- **ProcessRaise**:
  - `RaiseAmount = BigBlindAmount * 2` (고정 금액)
  - `CurrentBet += RaiseAmount`
  - `CurrentMaxBet = Player->CurrentBet` (최고액 갱신)
  - **중요**: 다른 플레이어들의 `bHasActedThisRound = false` 리셋 (한 바퀴 더 돌아야 함)
- **CollectBetsIntoPot**:
  - `TotalPot += Player->CurrentBet` (팟에 합산)
  - `TotalBet += CurrentBet` (누적 베팅액)
  - `CurrentBet = 0` (라운드 리셋)
  - `CurrentMaxBet = 0`

#### 3. GameMode 베팅 라운드 관리 함수 구현 (3개)

**HoldemGameMode.h/cpp 추가:**

```cpp
void StartBettingRound();
bool IsBettingRoundComplete();
void MoveToNextPlayer();
```

**주요 로직:**
- **StartBettingRound()**:
  - 모든 플레이어의 `bHasActedThisRound = false` 리셋
  - `CurrentTurnPlayerIndex = 0` (첫 플레이어 설정, 추후 Dealer/Blind 고려)
  - Fold한 플레이어 건너뛰기

- **IsBettingRoundComplete()**:
  - 조건 1: 모든 Active 플레이어가 `bHasActedThisRound == true`
  - 조건 2: 모든 Active 플레이어의 `CurrentBet == CurrentMaxBet`
  - 예외: Active 플레이어가 1명 이하면 즉시 종료 (모두 Fold)

- **MoveToNextPlayer()**:
  - `IsBettingRoundComplete()` 체크
  - 완료되면: `CollectBetsIntoPot()` 호출 → 다음 Phase로 전환
  - 미완료: 다음 플레이어로 턴 이동 (Fold한 플레이어 스킵)

**이유:**
- `CurrentTurnPlayerIndex`: PlayerArray의 인덱스 (0, 1, 2, 3...)
- `GetPlayerId()`: 언리얼 내부 고유 ID (큰 숫자)
- 두 값을 직접 비교하면 항상 다르게 나옴!
