#### 2. HandEvaluatorTester 구현

**파일 구조**
- Public/Tester/HandEvaluatorTester.h
- Private/Tester/HandEvaluatorTester.cpp

**FPlayerTestData 구조체**
- PlayerName (FString): 플레이어 식별용
- HandCards (TArray<FCardData>): 손패 2장

**기능 1: EvaluateHand()**
- 입력: Cards (TArray<FCardData>) - 7장
- 출력: Test1Result (FString)
- 로직:
  1. 입력 검증 (정확히 7장)
  2. FCardData → 임시 ACard 액터 스폰
  3. HandEvaluator->GetBestHandFrom7Cards() 호출
  4. 결과를 문자열로 변환 (HandEvaluationToString)
  5. 임시 액터 정리

**기능 2: DetermineTestWinner()**
- 입력:
  - CommunityCards (TArray<FCardData>) - 5장
  - PlayerTestData (TArray<FPlayerTestData>) - 각 플레이어 손패 2장
- 출력: Test2Result (FString)
- 로직:
  1. 입력 검증 (최소 2명, 공개 5장, 손패 2장)
  2. GetWorld()->GetGameState()로 HoldemGameState 가져오기
  3. GameState->CommunityCards 임시 설정 (원본 백업)
  4. NewObject<AHoldemPlayerState>()로 가짜 PlayerState 생성
  5. 각 PlayerState의 HandCards에 손패 할당
  6. **GameState->DetermineWinner(TestPlayers) 직접 호출**
  7. 승자 결과 출력 (단독/동점 구분)
  8. 정리 및 원상 복구

**헬퍼 함수**
- HandEvaluationToString(): FHandEvaluation → "TwoPair [14, 14, 13, 13, 10]"
- HandRankToString(): EHandRank → "Two Pair"
