## 테스트 케이스

### 테스트 케이스 형식 안내

**FCardData 생성 방법:**
```cpp
FCardData(ECardSuit::Spades, ECardRank::Ace)  // ♠A
FCardData(ECardSuit::Hearts, ECardRank::King)  // ♥K
FCardData(ECardSuit::Diamonds, ECardRank::Queen)  // ♦Q
FCardData(ECardSuit::Clubs, ECardRank::Jack)  // ♣J
```

**ECardRank 참고:**
- Ace = 0 (값: 14)
- Two = 1 (값: 2)
- Three = 2 (값: 3)
- Four = 3 (값: 4)
- Five = 4 (값: 5)
- Six = 5 (값: 6)
- Seven = 6 (값: 7)
- Eight = 7 (값: 8)
- Nine = 8 (값: 9)
- Ten = 9 (값: 10)
- Jack = 10 (값: 11)
- Queen = 11 (값: 12)
- King = 12 (값: 13)

## Test2: 승자 판정 테스트

### 승자 판정 케이스 1: 3명의 플레이어 - 명확한 승자
**시나리오:** Player1이 스트레이트로 승리

**공개 카드 (CommunityCards):**
```cpp
CommunityCards = {
    FCardData(ECardSuit::Hearts, ECardRank::King),   // ♥K
    FCardData(ECardSuit::Diamonds, ECardRank::Queen), // ♦Q
    FCardData(ECardSuit::Clubs, ECardRank::Jack),    // ♣J
    FCardData(ECardSuit::Spades, ECardRank::Ten),    // ♠10
    FCardData(ECardSuit::Hearts, ECardRank::Two)     // ♥2
};
```

**플레이어 데이터 (PlayerTestData):**

**Player 1 - "Alice"**
```cpp
PlayerTestData[0].PlayerName = "Alice";
PlayerTestData[0].HandCards = {
    FCardData(ECardSuit::Spades, ECardRank::Ace),    // ♠A
    FCardData(ECardSuit::Diamonds, ECardRank::Three) // ♦3
};
// 최종 핸드: A-K-Q-J-10 (스트레이트)
```

**Player 2 - "Bob"**
```cpp
PlayerTestData[1].PlayerName = "Bob";
PlayerTestData[1].HandCards = {
    FCardData(ECardSuit::Hearts, ECardRank::King),   // ♥K
    FCardData(ECardSuit::Clubs, ECardRank::King)     // ♣K
};
// 최종 핸드: K-K-K-Q-J (트리플 킹)
```

**Player 3 - "Charlie"**
```cpp
PlayerTestData[2].PlayerName = "Charlie";
PlayerTestData[2].HandCards = {
    FCardData(ECardSuit::Spades, ECardRank::Nine),   // ♠9
    FCardData(ECardSuit::Hearts, ECardRank::Eight)   // ♥8
};
// 최종 핸드: K-Q-J-10-9 (하이카드)
```

