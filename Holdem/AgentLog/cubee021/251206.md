#### 1. 베팅 시스템 아키텍처 설계 완료

**베팅 라운드 종료 조건 (방법 A 선택):**
- `bHasActedThisRound` (각 플레이어가 액션했는지)
- `CurrentMaxBet` (현재 라운드 최고 베팅액)
- Raise 발생 시 다른 플레이어들의 `bHasActedThisRound = false`로 리셋
- 모든 Active 플레이어가 `bHasActedThisRound == true && CurrentBet == CurrentMaxBet`이면 종료

**칩 시스템 구조 확정:**
- `CurrentChips`: 플레이어가 보유한 총 칩 (UI "Current Chips")
- `CurrentBet`: 현재 베팅 라운드에 낼 금액 (UI "Bet")
- `TotalBet`: 이번 판(Hand)에 베팅한 총액 (UI "Total Bet")
- 베팅 라운드 종료 시: `TotalBet += CurrentBet`, `CurrentBet = 0`

**Blind 처리:**
- PreFlop: BigBlind 다음부터 시작 → BigBlind로 끝
- Flop/Turn/River: SmallBlind부터 시작 → Dealer로 끝
- **중요**: Blind 강제 베팅은 Fold해도 회수 불가 (이미 팟에 기여)

**칩 제한:**
- `StartingChips` 제거 (회원가입 시 지급)
- `MinimumChips = 100` 추가 (테이블 입장/유지 최소 칩)
- Side Pot은 추후 구현

**UI 버튼 구조:**
- 3개 버튼 고정 슬롯
- 상황별 매핑:
  - `CurrentMaxBet == 0`: [Bet, Check, Fold]
  - `CurrentBet < CurrentMaxBet`: [Raise, Call, Fold]
  - `CurrentBet == CurrentMaxBet && MaxBet > 0`: [Raise, Check, Fold]

```cpp
}
```

```cpp
void UMyPlayerWidget::UpdateButtonTexts()
{
    bool bNoBets = (GS->CurrentMaxBet == 0);
    bool bCanCheck = (PS->CurrentBet == GS->CurrentMaxBet);

    // Button 0
    if (bNoBets)
        Txt_0->SetText(FText::FromString(FString::Printf(TEXT("Bet %d"), GS->BigBlindAmount)));
    else
        Txt_0->SetText(FText::FromString(FString::Printf(TEXT("Raise %d"), GS->BigBlindAmount * 2)));

    // Button 1
    if (bCanCheck)
        Txt_1->SetText(FText::FromString(TEXT("Check")));

    // Button 2
    Txt_2->SetText(FText::FromString(TEXT("Fold")));
}
```
