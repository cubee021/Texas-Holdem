#### 1. 참고 게임 분석 → Fixed Limit Hold'em 5/10 구조 파악

**발견한 베팅 구조:**
- **Blind**: Small Blind = 2, Big Blind = 5
- **Betting Limits**: PreFlop/Flop = 5, Turn/River = 10
- **Raise 방식**: CurrentMaxBet + BettingUnit
- **추가로 내야 할 금액**: (NewMaxBet - CurrentBet)

**예시:**
```
Small Blind (CurrentBet=2)가 MaxBet=10일 때 Raise:
- NewMaxBet = 10 + 5 = 15
- AdditionalAmount = 15 - 2 = 13  ← 이게 "Raise 13"!
```

- No-Limit 방식으로 구현됨 (BigBlindAmount * 2)
- Blind 자동 베팅 없음

```cpp
void AHoldemGameState::ProcessRaise(AHoldemPlayerState* Player)
{
    // Fixed Limit: 현재 Phase에 따른 베팅 단위 결정
    int32 BettingUnit = (CurrentPhase == EHoldemPhase::Turn ||
                         CurrentPhase == EHoldemPhase::River)
                         ? BigBetAmount : SmallBetAmount;

    // Raise 후 새로운 MaxBet 계산
    int32 NewMaxBet = CurrentMaxBet + BettingUnit;

    // 내가 추가로 내야 할 금액
    int32 AdditionalAmount = NewMaxBet - Player->CurrentBet;

    Player->CurrentChips -= AdditionalAmount;
    Player->CurrentBet = NewMaxBet;
    // ...
}
```

#### 3. 멀티플레이 테스트 중 발견한 버그 수정

**버그 1: static 변수로 인한 플레이어 간 공유 문제**

**증상:**
```
Player 0이 버튼 누름 → LastConfirmTime 업데이트
Player 1이 바로 버튼 누름 → "Blocked duplicate input!" ← 잘못 막힘!
```

**원인:**
```cpp
// MyPlayerController.cpp
static float LastConfirmTime = 0.f;  // ❌ 모든 PlayerController가 공유!
```

**해결:**
- MyPlayerController.h에 멤버 변수로 변경:
```cpp
protected:
    float LastConfirmTime = 0.f;  // ✅ 각 PlayerController마다 독립적
```

**버그 2: SelectedButtonIndex와 UI 표시 불일치**

**증상:**
```
화면에는 Button 1 (Check/Call)이 노란색으로 표시되는데
실제로는 Button 0 (Raise)가 실행됨
```

**원인:**
```cpp
// MyPlayerWidget.cpp - OnTurnChangedHandler()
void UMyPlayerWidget::OnTurnChangedHandler(int32 NewTurnIndex)
{
    // ...
    if (PC && PC->IsMyTurn())
    {
        OnBettingSelectionChanged(1);  // ❌ UI만 업데이트!
        // PC->SelectedButtonIndex는 변경 안됨!
    }
}
```

**해결:**
```cpp
void UMyPlayerWidget::OnTurnChangedHandler(int32 NewTurnIndex)
{
    // ...
    if (PC && PC->IsMyTurn())
    {
        PC->SelectedButtonIndex = 1;  // ✅ 실제 변수도 리셋!
        OnBettingSelectionChanged(1);
        UpdateButtonTexts();
    }
}
```
